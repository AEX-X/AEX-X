<h2 align="center">О СЕБЕ</h2>

<p align="center">
<b>Беру идею и довожу её до рабочего, задеплоенного MVP сам — бэкенд, фронтенд и инфраструктура.</b><br/>
Fullstack-инженер с продуктовым мышлением, из Москвы.
</p>

- **Самоучка** — примерно за год прошёл путь с нуля до запуска **распределённой, устойчивой к цензуре VPN-платформы**.
- Строю **end-to-end**: бэкенд, фронтенд, автоматизация инфраструктуры и продуктовая логика, которая заставляет продукт реально конвертить.
- Уверенно работаю с **async Python + FastAPI**, **платформами Telegram** (боты и WebApp'ы), **Docker / CI** и **автоматизацией инфраструктуры через SSH**.
- Отношусь к AI как к **ежедневному рабочему инструменту** — быстро выкатываю на OpenAI / Claude API и AI-assisted разработке.
- **Сейчас прокачиваю:** Cloud & DevOps — Kubernetes, CI/CD, observability.
- **Открыт к командам на хакатонах** — могу вести бэкенд, фронтенд или довести весь MVP до деплоя в одиночку.
